# Soviet Wagon - 3D Environment Case Study 🚂

*Created for SoiGames Case Study*

A highly optimized, game-ready 3D environment built in Unity. This project demonstrates a complete workflow from asset organization to baked lighting and dynamic probe setups, culminating in a playable First-Person (FPS) walkthrough.

## 🛠️ Technical Details
- **Unity Version:** `6000.3.9f1`
- **Delivery Format:** GitHub Repository (LFS Enabled)

## ✨ Key Features
* **Game-Ready Optimization:** Clean hierarchy, prefabs, and zero console errors.
* **Advanced Lighting:** Baked Global Illumination (GI) for the static wagon environment for maximum performance.
* **Dynamic Integration:** Configured `Light Probe Groups` to ensure moving characters (`businessman` and Player `Arms`) realistically react to the baked environment lighting.
* **Playable Scene:** Included a custom FPS Controller so reviewers can easily walk around and inspect the environment (W-A-S-D to move, Mouse to look).
* **Modular Prefabs:** Separated the environment (`Vagon_Environment`), NPCs (`businessman`), and Player (`Arms`) into distinct, reusable prefabs.

## 🚀 How to Run
1. Clone or download this repository.
2. Open the project using **Unity 6000.3.9f1**.
3. Navigate to `Assets/Scenes/` and open the main scene (e.g., `Vagon_Environment_1.unity`).
4. Press the **Play** button at the top of the Unity Editor.
5. Use your Mouse to look around and W-A-S-D keys to walk through the wagon.
